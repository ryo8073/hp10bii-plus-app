# HP10BII+ 電卓エミュレーターと資本蓄積計算ツール ワイヤーフレーム

## 1. 全体構造

アプリケーションは以下の2つの主要セクションで構成されます：

1. HP10BII+ 電卓エミュレーター
2. 資本蓄積計算ツール

これらはタブ切り替えで利用できるようにします。

```
+------------------------------------------+
|  [HP10BII+ 電卓] | [資本蓄積計算]        |
+------------------------------------------+
|                                          |
|            コンテンツエリア              |
|                                          |
+------------------------------------------+
```

### 2.2 キー機能

各キーには以下の情報が表示されます：

- メインの機能（中央）
- シフトアップ機能（青色、左上）
- シフトダウン機能（オレンジ色、右下）

```
+---------------------+
| シフトアップ機能    |
|                     |
|   メイン機能        |
|                     |
|           シフトダウン機能 |
+---------------------+
```

### 2.3 ディスプレイ

```
+------------------------------------+
| [シフト] [モード:FIN] [メモリ:M]   |
| 0                                  |
+------------------------------------+
```

- シフト表示：現在のシフト状態（↑または↓）
- モード表示：現在のモード（FIN, STAT, など）
- メモリ表示：メモリ使用状態（M）
- 数値表示：計算結果や入力値

### 3.3 入力フォーム

- 再投資率（%）：資本蓄積に回す割合
- 安全率（%）：安全資産として確保する割合
- 総投資額（円）：初期投資額
- 計算年数：何年分計算するか

## 4. 共通UI要素

### 4.1 ナビゲーション

```
+------------------------------------------+
|  [HP10BII+ 電卓] | [資本蓄積計算]        |
+------------------------------------------+
```

- タブ形式のナビゲーション
- アクティブなタブは視覚的に区別

### 4.2 レスポンシブデザイン

- デスクトップ：横向きレイアウト、大きなキーと表示
- タブレット：適応型レイアウト、中程度のキーと表示
- モバイル：縦向きレイアウト、タップしやすいキーサイズ

### 4.3 ダークモード

- システム設定に応じて自動的に切り替え
- 適切なコントラスト比を維持

## 5. アクセシビリティ

- キーボードナビゲーション
- スクリーンリーダー対応
- 十分なコントラスト比
- フォーカス表示
